Perguntas e Respostas; Link para o formulário a baixo:

  1. Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

    Atributo "placeholder" na tag < input> com type="text": O atributo "placeholder" é utilizado para fornecer um texto de exemplo ou uma dica sobre o que deve ser inserido no campo de texto. Ele geralmente descreve o formato ou o tipo de dados esperado.

  2. Na tag < input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

    Atributo "checked" nas tags < input> com type="radio" ou type="checkbox": O atributo "checked" é utilizado para pré-selecionar um botão de opção (radio) ou uma caixa de seleção (checkbox) por padrão. Isso significa que, quando a página é carregada, esse botão ou caixa estará marcado automaticamente.

  3. Na tag < input para que serve o atributo "value"?

    Atributo "value" na tag < input>: O atributo "value" é utilizado para definir o valor inicial ou o valor que será enviado quando o formulário for submetido. No caso de campos de texto, é o valor que será inserido inicialmente no campo.

  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

    Atributo para agrupar botões de rádio em um formulário: O atributo "name" é utilizado para agrupar vários botões de rádio em um formulário, garantindo que apenas um deles possa ser selecionado de cada vez.

  5. Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

    Tag < label> e atributo "for": A tag < label> é usada para rotular um elemento de formulário, como um campo de entrada. O atributo "for" na tag < label> está associado ao ID do elemento de formulário que está sendo rotulado, melhorando a acessibilidade e permitindo que os usuários cliquem no rótulo para focar no campo correspondente.

  6. Na tag < option > para que serve o atributo "selected"?

    Atributo "selected" na tag < option>: O atributo "selected" é utilizado para especificar qual opção de um elemento < select> deve ser selecionada por padrão quando a página é carregada.

  7. Nas tags de formulário para que serve o atributo "required"?

    Atributo "required" nas tags de formulário: O atributo "required" é utilizado para indicar que um campo de formulário deve ser preenchido antes que o formulário possa ser enviado. Ele é usado para garantir que certas informações sejam fornecidas pelo usuário.

  8. Para que serve a tag < textarea >?

    Tag < textarea>: A tag < textarea> é usada para criar uma área de texto multi-linhas em um formulário, permitindo que os usuários insiram textos mais longos.

  9. Na tag < form > para que serve o atributo "action"?

    Atributo "action" na tag < form>: O atributo "action" na tag < form> especifica para onde os dados do formulário serão enviados quando o formulário for submetido.

  10. Na tag < form > para que serve o atributo "method"?

    Atributo "method" na tag < form>: O atributo "method" na tag < form> especifica o método HTTP a ser usado ao enviar os dados do formulário. Pode ser "GET" ou "POST".

  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form>?

    Para ocultar senha na barra de endereços: Não é o atributo "method", mas o atributo "method" deve ser definido como "POST" para ocultar a senha na barra de endereços.

  12. Qual valor deve ser informado no atributo "method" na tag < form> em um formulário que será usado para fazer upload de arquivos?

    Atributo "method" para upload de arquivos: Para formulários que serão usados para fazer upload de arquivos, o valor do atributo "method" deve ser "POST".

  13. O que as tags < fieldset> e < legend> produzem?

    Tags < fieldset> e < legend>: A tag < fieldset> é usada para agrupar elementos de formulário relacionados, e a tag < legend> fornece uma legenda ou título descritivo para o grupo criado pelo < fieldset>. Isso é útil para organizar e tornar mais compreensíveis os formulários mais extensos.
